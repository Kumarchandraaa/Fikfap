About This Site

This site is hosted on GitHub Pages, which is a free hosting service for static websites.

Feel free to explore and learn more about how GitHub Pages works!

What is GitHub Pages?

GitHub Pages is a free web hosting service that allows you to host static websites directly from a GitHub repository.

How to Use GitHub Pages?

1. Create a new repository with the name `username.github.io`.

2. Upload your HTML, CSS, and other files to that repository.

3. Enable GitHub Pages in your repository's settings, and your website will be live!
